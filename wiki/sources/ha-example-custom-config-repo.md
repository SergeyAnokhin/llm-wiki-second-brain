# HA Example Custom Config Repository

**Source:** `Repository of official custom_components · home-assistantexample-custom-config.md`
**URL:** https://github.com/home-assistant/example-custom-config
**Type:** reference page (GitHub repository)

## Summary

Official Home Assistant repository of example custom configurations. Files are organized like a real HA config directory — copy any file to the same path in your own config to use it.

## Key Claims

- Covers two extension types: **custom components** (loaded via `configuration.yaml`, same as built-in) and **custom panels** (via `panel_custom` component)
- Serves as an official reference for how custom components should be structured
