# Contributing to Home Assistant

**Source:** `CONTRIBUTING.md`
**URL:** https://github.com/home-assistant/core
**Type:** contribution guide

## Summary

Brief guide for contributing to HA core. Describes the PR process and where to report issues or suggest features.

## Key Claims

- Fork `home-assistant/core`, write code, ensure tests work, create PR against the **dev** branch (not main)
- Feature suggestions: start a discussion at https://github.com/orgs/home-assistant/discussions
- Bug reports: create an issue at https://github.com/home-assistant/core/issues
- Read [How to get faster PR reviews](https://github.com/kubernetes/community/blob/master/contributors/guide/pull-requests.md) (Kubernetes guide, skip steps 0 and 1)
